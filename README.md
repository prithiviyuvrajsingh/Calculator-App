# Calculator-App
A professional BMI Calculator built with React.js that calculates Body Mass Index based on height and weight. It features a modern responsive UI, gradient background, color-coded BMI status, health suggestions, reset functionality, and clean user experience.
# Calculator App

A simple and responsive calculator built using ReactJS.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Clear button
- Responsive design

## Technologies Used

- ReactJS
- JavaScript
- CSS

## Run Locally

```bash
npm install
npm start
```
